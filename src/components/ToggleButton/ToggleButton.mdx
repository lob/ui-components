A toggle button is a styled version of our checkbox.

## How to Use

This toggle button component is designed to be used on its own in the case of a simple true/false or within a list to indicate a selection of options.

The label and v-model are required. The other props are optional.

To use this toggle button as a simple boolean, here is an example

```html
<toggle-button
  label="I am a toggle button"
  v-model="mySwitch"
>
</toggle-button>
```

To use this toggle button in a list to indicate an array of selected options, use the same v-model for all buttons. It will be an array of strings.

Here is an example

```html
<div>
    <toggle-button
      label="Chocolate"
      v-model="selectedIceCreamFlavors"
      name="chocolate"
      value="chocolate"
    />
    <toggle-button
      label="Vanilla"
      v-model="selectedIceCreamFlavors"
      name="vanilla"
      value="vanilla"
    />
    <toggle-button
      label="Strawberry"
      v-model="selectedIceCreamFlavors"
      name="strawberry"
      value="strawberry"
    />
  </div>
```
