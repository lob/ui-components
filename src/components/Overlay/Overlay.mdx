# Overlay

<Canvas of={Primary} />

## Usage

1. Create a template ref with the `Overlay` component

```html
<script>
  const overlayRef = ref();
</script>

<template>
  <Overlay ref="overlayRef" />
</template>
```

2. Use the `toggle` method bound with an event to toggle the `Overlay`

```html
<script>
  const overlayRef = ref();
</script>

<template>
  <button @click="(e) => overlayRef.toggle(e)">Toggle Overlay</button>
  <Overlay ref="overlayRef" />
</template>
```
