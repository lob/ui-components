# Checkbox

A checkbox represents a value a form that can either be true or false.

  <Story id="components-checkbox--single" />
</Preview>

## How to Use

This checkbox component is designed to be used on its own in the case of a simple true/false or within a list to indicate a selection of options.

The label and v-model are required. The other props are optional.

To use this checkbox as a simple boolean, here is an example

```html
<checkbox
  label="I am a checkbox"
  v-model="mySwitch"
>
</checkbox>
```

To use this checkbox in a list to indicate an array of selected options, use the same v-model for all checkboxes. It will be an array of strings.

Here is an example

```html
<div>
    <checkbox
      label="Chocolate"
      v-model="selectedIceCreamFlavors"
      name="chocolate"
      value="chocolate"
    />
    <checkbox
      label="Vanilla"
      v-model="selectedIceCreamFlavors"
      name="vanilla"
      value="vanilla"
    />
    <checkbox
      label="Strawberry"
      v-model="selectedIceCreamFlavors"
      name="strawberry"
      value="strawberry"
    />
  </div>
```
