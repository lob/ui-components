# Card

Cards are a flexible component used to display content in a variety of contexts.

<Canvas of={Primary} />

## How to Use

Please note: though you can only insert text in the Storybook demo, the card component will accept any valid HTML or Vue component.

```html
<card>
  <p>I am a card.</p>
</card>
```
