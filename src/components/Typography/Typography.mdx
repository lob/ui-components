# Typography

This component provides a consistent set of text styles.

Individual components for each style are available (`<H1 />`, `<P />`, `<Caption />`, etc.)

<Preview>
  <Story id="components-typography--lorem-ipsum" />
</Preview>

## How to Use

```html
<Typography variant="heading1">
  This is a heading
</Typography>
<H1>
  This is also a heading
</H1>
<P>
	This is body text
</P>
```
